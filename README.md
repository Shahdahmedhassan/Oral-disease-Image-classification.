# 🦷 Dental Disease Image Classification

A deep learning app that classifies oral/dental conditions from an uploaded image.

## 🔗 Try the live app
👉 [Click here to open the app](https://oral-disease-image-classification-yxpf8eoh7lwmxeg8fjlalp.streamlit.app)

## About
This app uses a trained CNN/transfer-learning model to detect common oral 
diseases such as Calculus, Gingivitis, Caries, Hypodontia, Mouth Ulcer, 
and Tooth Discoloration from a single image.
